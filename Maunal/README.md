# Maunal — Manual Findings & Reports

> This directory stores ad-hoc analysis, findings, and reports generated by the `/orion` command.

## Structure
- `CONTEXT.md` — Persistent context tracking all Orion sessions
- `ORION-NNN/` — Per-session analysis folders

## Usage
```
/orion @path/to/file-or-folder
```

## Output Types
| Type | When Created |
|------|-------------|
| FINDINGS | Issues or failures identified |
| SOLUTIONS | Actionable fixes recommended |
| REPORT | Status comparison needed |
| TASKS | Discrete work items |
| PLAN | Phased fix approach |
